# octranspo

This is just a small CLI that lists the upcoming trips for a given OC Transpo
stop. For example:

```
$ octranspo stop 845
62 Kanata (Outbound): no upcoming trips.
96 Kanata Terry Fox (Westbound):
  20:36 (Terry Fox)
  20:49 (Stittsville)
  21:06 (Terry Fox)
168 Beaverbrook (Northbound): no upcoming trips.
```
